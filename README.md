# Hometown Electrical Website

A professional, modern website for an electrical services business based in Melbourne. This website is designed to attract customers searching for electrical services in Melbourne and surrounding suburbs, with a focus on SEO, modern design, and user experience.

## Features

- **Bold, Simple, Stylish UI**: Clean, modern design with a focus on conversions
- **SEO Optimized**: Meta tags, schema markup, and semantic HTML
- **Mobile Responsive**: Fully responsive design for all devices
- **Smooth Scroll Animations**: Elements animate into view as the user scrolls
- **Interactive Elements**: 
  - FAQ accordion
  - Testimonial slider
  - Mobile-friendly navigation
- **Call-to-Action Buttons**: Strategically placed to maximize conversions
- **Service Area Listings**: Targeted location-based content for local SEO
- **Contact Form**: Easy to use quote request form
- **Floating CTA Buttons**: Always visible call and quote buttons

## SEO Features

- **Schema.org Markup**: Structured data for better search engine understanding
- **Meta Tags**: Optimized title, description and keywords
- **Open Graph Tags**: For better social media sharing
- **Semantic HTML**: Properly structured content with appropriate heading levels
- **Location-Based Content**: Service areas specified for local SEO
- **Fast Loading**: Optimized images and code for speed
- **Mobile Friendly**: Google prioritizes mobile-friendly websites

## Technologies Used

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript (no dependencies)
- Font Awesome Icons
- Google Fonts

## Setup Instructions

### Prerequisites

- Web hosting service
- Domain name (preferably something with "Melbourne" and "electrician" for SEO)

### Installation

1. Clone or download this repository
2. Upload all files to your web hosting service
3. Create/prepare the following directories and files:
   - `images/` directory with:
     - `logo2.png` (your company logo)
     - `logo-white.png` (white version for footer)
     - `logo-flavicon.png` (site icon)
     - `og-image.jpg` (image for social sharing)
     - `hero-bg.jpg` (background image for hero section)
     - Service images (emergency-repairs.jpg, installations.jpg, etc.)
   
4. Update the following information in the HTML files:
   - Business name and contact information
   - Service areas (Melbourne suburbs)
   - Services offered
   - Customer testimonials
   - Business hours
   - Google Maps integration (if desired)
   
5. Update schema.org markup in the `<head>` section with your business details
6. Set up form handling for the contact/quote form (update the form action in the HTML)

### Customization

- Colors can be easily changed by editing the CSS variables in the `:root` section of `styles.css`
- Fonts can be changed by updating the Google Fonts import and font family variables
- Additional pages can be created by copying the existing template and updating content

## SEO Recommendations

- Register your business with Google My Business
- Build citations on local directories (Yellow Pages, Yelp, TrueLocal, etc.)
- Create location-specific content for each major Melbourne area you serve
- Implement a blog with helpful electrical tips and advice
- Encourage satisfied customers to leave Google reviews
- Consider adding case studies of completed electrical projects
